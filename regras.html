Regras do Jogo

O jogo Whac-A-Mole é um jogo de habilidade e reflexos onde o objetivo principal é acertar o maior número possível de toupeiras, que irão aparecer aleatoriamente em diferentes buracos no tabuleiro.

Durante o jogo, as toupeiras surgem por um curto período de tempo, o jogador deve ser rápido para clicar nelas antes que desapareçam. O jogo esta divido em 3 niveis: - Modo Fácil: Para cada toupeira acertada, o jogador ganha 5 pontos. Cada toupeira perdida são 2 pontos a menos, e para cada topeira errada são 3 pontos a menos. - Modo Médio: Para cada topeira acertada, o jogador ganhar 10 pontos. Cada toupeira perdida são 4 pontos a menos, e para cada topeira errada são 6 pontos a menos. - Modo Difícil: Para cada topeira acertada, o jogador ganhar 20 pontos. Cada toupeira perdida são 8 pontos a menos, e para cada topeira errada são 12 pontos a menos. O jogo termina quando o tempo acaba.
